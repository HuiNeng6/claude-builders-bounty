# claude-review

AI-powered PR reviewer using Claude Code.

## Usage (CLI)

```bash
# Install Claude Code
npm install -g @anthropic-claude/claude-code

# Review a PR
claude-review --pr https://github.com/owner/repo/pull/123
```

## Usage (GitHub Action)

Create `.github/workflows/pr-review.yml`:

```yaml
name: PR Review Bot
on: pull_request
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: claude-builders-bounty/pr-review@v1
        with:
          anthropic_api_key: ${{ secrets.ANTHROPIC_API_KEY }}
```

## Output Format

```markdown
## Summary
This PR adds user authentication with NextAuth.js and includes login/signup pages. The implementation follows Next.js App Router patterns.

## Identified Risks
- No CSRF protection on form submissions
- Missing rate limiting on auth endpoints
- Password stored in plain text (should hash)

## Improvement Suggestions
- Add CSRF token validation
- Implement rate limiting (max 5 attempts per minute)
- Use bcrypt for password hashing

## Confidence Score
Medium

---
Generated by claude-review
```

## Requirements

- Claude Code CLI installed
- GitHub CLI (gh) installed
- ANTHROPIC_API_KEY environment variable

```bash
git clone https://github.com/HuiNeng6/claude-review.git
chmod +x claude-review.sh && ./claude-review.sh --pr https://github.com/owner/repo/pull/123
```
