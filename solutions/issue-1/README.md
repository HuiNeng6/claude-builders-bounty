# changelog-generator

Generate a structured CHANGELOG from git history automatically.

## Installation (2 commands)

```bash
git clone https://github.com/HuiNeng6/changelog-generator.git
chmod +x changelog-generator/changelog.sh && ./changelog-generator/changelog.sh
```

## Usage

```bash
# Generate for current repo
./changelog.sh

# Generate for specific repo
./changelog.sh /path/to/repo

# Generate for specific date range
./changelog.sh . 2024-01-01 2024-12-31
```

## Output

Creates a `CHANGELOG.md` file with:

- Version sections (by git tags)
- Categories: Added, Changed, Fixed, Removed
- Commit messages grouped by type
- Authors attribution

## Example Output (tested on claude-builders-bounty)

```markdown
# CHANGELOG

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2024-12-01

### Added
- feat: Add new user authentication system (John Doe)
- Add password reset feature (Jane Smith)

### Fixed
- fix: Resolve login timeout issue (John Doe)

### Changed
- update: Improve API response time (Jane Smith)

### Generated by changelog-generator
### 2026-03-28
```

## Requirements

- Git installed
- Bash shell
- A git repository with commit history
